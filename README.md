
# OSINT Tools Directory

A curated collection of Open Source Intelligence (OSINT) tools for cybersecurity professionals, investigators, and researchers.

## Features

- 🔍 **Search & Filter** — Find tools quickly by name, description, or category
- 📂 **Category Navigation** — Browse tools organized by functionality
- 🎨 **Cyber-themed UI** — Modern, responsive interface with dark mode aesthetics
- 📱 **Mobile Friendly** — Fully responsive design for all devices

## Tech Stack

- **React** + **TypeScript**
- **Tailwind CSS**
- **Lucide React** (icons)
